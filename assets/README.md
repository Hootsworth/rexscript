# RexScript Assets

This folder is designed for README production assets.

## Structure

- `code-chunks/`: Source snippets prepared for CodeSnap captures.
- `screenshots/`: Final image outputs referenced by README.

## Workflow

1. Open any file from `code-chunks/` in your editor.
2. Use CodeSnap to capture the snippet.
3. Save the image into `screenshots/` using the exact filename from `screenshot-manifest.md`.
4. Commit both code chunk and screenshot changes.

## Naming Rules

- Keep the numeric prefix unchanged.
- Use PNG format.
- Keep screenshot width readable for GitHub (`1600px` recommended).
